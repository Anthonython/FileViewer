itemAdapter - адаптер для отображения списка файлов;
currentDir - директория, в которой находятся отображаемые файлы (по умолчанию - корневая директория);
fileList - список файлов в текущей директории;
fileModels - список моделей файлов, используемый для заполнения адаптера;
sortType - тип сортировки файлов (по умолчанию - сортировка по имени);
hashFileList - список файлов, которые необходимо проверить на наличие изменений;
coroutineScope - область корутин;
hashCodes - список хеш-кодов файлов, хранящихся в базе данных;
db - экземпляр базы данных;
dbManager - менеджер для работы с базой данных;
files - виджет для отображения списка файлов.
Метод onCreate вызывается при создании активности. В данном методе происходит инициализация элементов интерфейса, установка слушателей для выпадающих списков, инициализация базы данных и вызов метода init.

Метод init проверяет наличие необходимых разрешений для работы с файлами, а затем вызывает метод updateList с корневой директорией.

Метод setIcon возвращает идентификатор иконки для файла на основе его расширения и типа (файл или папка).

Метод checkHash проверяет хеш-коды файлов на изменения и возвращает список файлов, которые были изменены.

Метод updateList обновляет список файлов в текущей директории и обновляет адаптер для отображения списка.

Метод onBackPressed переопределяет стандартное поведение кнопки "Назад" и вызывает метод updateList с родительской директорией.


Стек используемых технологий в коде:

Kotlin - язык программирования, на котором написан код.
Android SDK - набор инструментов разработки для создания приложений под Android.
AndroidX - библиотека компонентов Android, используемых в разработке приложений.
Material Design - система дизайна, используемая для создания пользовательских интерфейсов Android приложений.
RecyclerView - виджет для отображения списков в приложении.
Spinner - виджет, представляющий собой раскрывающийся список с выбором одного из нескольких элементов.
Room - библиотека для работы с базами данных SQLite в Android приложениях.
Coroutines - библиотека для управления многопоточностью и асинхронными операциями в Kotlin.
Dispatchers - класс для управления потоками в библиотеке Coroutines.
SupervisorJob - класс для управления ошибками в библиотеке Coroutines.
AppCompatActivity - базовый класс для создания активити в Android приложениях.
